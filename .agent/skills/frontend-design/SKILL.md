# Frontend Design

Create distinctive, production-grade frontend interfaces that avoid generic "AI slop" aesthetics.

## Input Types

### Screenshot/Image Provided
1. **Extract Design Guidelines** using `references/design-extraction-overview.md`
2. **Implement Code** following extracted guidelines (exact colors, typography, spacing)
3. **Verify Quality** using `references/visual-analysis-overview.md`

### Building from Scratch
Follow Design Thinking process — commit to a BOLD aesthetic direction.

## Aesthetics Guidelines

- **Typography**: Distinctive font choices, not generic (avoid Arial, Inter)
- **Color**: Cohesive palette with sharp accents, CSS variables for consistency
- **Motion**: Meaningful animations, scroll-triggered reveals, hover surprises
- **Spatial**: Unexpected layouts, asymmetry, overlap, generous negative space
- **Backgrounds**: Atmosphere and depth — gradients, noise, patterns, shadows
