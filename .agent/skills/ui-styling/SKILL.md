# UI Styling

shadcn/ui components + Tailwind CSS + canvas-based visual design.

## Quick Start

```bash
npx shadcn@latest init          # Setup shadcn/ui + Tailwind
npx shadcn@latest add button card dialog form
```

1. Build complex UIs from composable primitives
2. Utility-first styling; extract only for true repetition
3. Mobile-first responsive design
4. Accessibility-first with Radix UI primitives
5. Consistent design tokens across the app
